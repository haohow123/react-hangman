# react-hangman

hangman web app created by vite+React+xstate

## Prerequisites

- nodeJs
- yarn

## Getting start

Clone project

```sh
git clone
```

Install it run:

```sh
yarn
yarn dev
```

## Dependencies

- **EmotionJs** - Writing css style with Javascript
- **xstate** - Javascript state machines and statecharts
